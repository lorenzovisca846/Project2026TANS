QUICK START

Come compilare ed eseguire il codice. Dalla cartella home del progetto:

    mkdir build
    cd build 
    cmake ..
    make

A questo punto il programma può essere eseguito dalla cartella build lanciando le tre fasi dell'esecuzione:

    ./simulation
    ./reconstruction
    ./analysis
